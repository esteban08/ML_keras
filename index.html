El Ciclo de Vida del Modelo

📌 El Problema

Entrenar un modelo es solo el primer paso. El verdadero desafío para un Ingeniero de Datos radica en cómo desplegar ese modelo para que usuarios sin conocimientos técnicos puedan utilizarlo en su día a día.

🎯 El Objetivo

Articular un modelo pre-entrenado (.h5) dentro de una interfaz web 100% funcional. Buscamos cerrar la brecha entre la ciencia de datos en su estado puro y la experiencia del usuario final.

⚙️ Funcionalidad

La aplicación desarrollada en Streamlit permite realizar inferencias en tiempo real procesando imágenes locales o utilizando la cámara web, emitiendo predicciones con su respectivo porcentaje de confianza.

Acerca del Desarrollo

Este proyecto fue estructurado aplicando prácticas de MLOps a nivel básico, asegurando que la transición desde el entorno de experimentación hasta el entorno productivo sea profesional y limpia.
